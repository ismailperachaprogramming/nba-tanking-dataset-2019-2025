# NBA Tanking Analysis (Post-2019 Lottery Reform)

> Inspired by the question: *Does tanking still work in the modern NBA?*

---

## Overview
The NBA updated its draft lottery system in 2019 to discourage teams from intentionally losing games (“tanking”) for better draft picks.

This project analyzes whether that reform actually worked.

We track lottery teams across multiple seasons and evaluate:
- Draft outcomes vs expectations  
- Pick ownership and conveyance  
- Future team success (playoffs, performance)


## Dataset Description

The dataset includes:
- Team performance (wins, win percentage, league rank)
- Lottery position and actual draft results
- Pick ownership (kept vs conveyed)
- Future success metrics (playoffs, win % over next 3 years)

See `data_dictionary.pdf` for full column definitions.

---

## Methodology

- Each row represents a **team-season for lottery teams**
- Focus is on **post-2019 lottery reform effects**
- Future success is measured over a **3-year window**
- Analysis uses only **observed outcomes** (no projections)
